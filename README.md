 ## Getting Started

 ### Prerequisites

- PHP
- WAMP / XAMP / UNISERVERZ / SERVER PHP
- PHPMYADMIN
 ### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/MaxenceR26/Intranet.git
   ```
3. Inject the SQL Database on your phpmyadmin.
4. Run ```index.html```
